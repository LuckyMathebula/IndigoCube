Cyber Security Development

Dynamic Application Security Testing (DAST) “black-box” DAST scanner sends mutated HTTP requests to a running app and examines how the app responds. Static Application Security Testing (SAST)“white-box” SAST Scanner examines application source code and traces data flow from ‘source’ to ‘sink’ to check if user input is sanitized. Open Source Software Security “OppSec” Discover vulnerabilities in open source components. Report on open source license violations. SAP Custom Code Analysis “Secure ABAB” Scan ABAB custom code stack for security vulnerabilities and quality defects (Performance, Reliability and Maintainability).
